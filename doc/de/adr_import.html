Importieren Sie tausende Adressen auf einmal. Wählen Sie eine CSV Datei die Sie auf den Server laden, oder wählen Sie eine vorhandene CSV-Datei aus der Sie Ihre Adressen importieren möchten. Die CSV-Datei muss folgendes Format besitzen: email,f0,f1,f2,f3,f4,f5,f6,f7,f8,f9 Als Trennzeichen kann ',' oder ';' oder '|' verwendet werden. Wählen Sie eine oder mehrere Gruppen denen die neuen Adressen zugeordnet werden sollen. Wählen Sie den Status den die importierten Adressen erhalten sollen jeweils für neue und bereits vorhandene Adressen, und ob diese ggf. aktiviert bzw de-aktivert werden sollen. (Im Zweifelsfalle belassen Sie den Status auf dem voreingestellten Wert ('manuell erfasst')) Existiert eine Adresse bereits in der Datenbank, so wird sie aktualisiert und den gewählten Gruppen zusätzlich zugeordnet. Um vorhandene Adressen in eine andere Gruppe zu 'verschieben', entfernen Sie das Häckchen bei 'Gruppen zusammenführen'. Alternativ können Sie auch die importierten Adressen löschen.
