 Danksagungen / Credits

Tellmatic ist ein OpenSource Projekt.

Open Source?
Tellmatic wurde und wird (hauptsächlich) in freier, unentgeltlicher Tätigkeit als freies Open Source Projekt entwickelt und weiter gepflegt.
Der Quelltext ist für jeden zugänglich und ohne Einschränkungen für private und kommerzielle Zwecke nutzbar.

Warum Spenden?
Spenden helfen dem Autor von Tellmatic das Projekt schneller voranzutreiben.
Das bringt wiederum den Nutzern ein verbessertes System.

Darum möchte ich alle Nutzer von Tellmatic bitten, einen Beitrag in Form einer freiwilligen Spende zu leisten.
Dienstleister, die mit Tellmatic kommerzielle Projekte realisieren, könnten einen kleinen Prozentsatz der Auftragssumme an das Tellmatic Projekt übermitteln.

Warum Ihre Spende?!?
Mit Ihrer Spende erreichen und bekommen Sie eine zeitnahere Fertigstellung der Software in einer überarbeiteten Version
und finanzieren dem Autor Zeit für konkrete Programmierung und Entwicklung an Code und Dokumentation.

Features sponsern
Zusätzlich besteht die Möglichkeit, gezielt Features zu sponsern um das System im eigenen Sinne weiterentwickeln zu lassen.
Die Ergebnisse hierbei - zum Vorteil aller Nutzer - fliessen dann in Form von Software-Updates in das Gesamtprojekt ein.

Beachten Sie bitte auch den  Installations- und Update-Service.


Wie spenden?
Die Spende erfolgt ganz einfach über Paypal.

Danksagungen / Credits
Die Sponsoren werden in einer Liste auf der Tellmatic Homepage unter  Danksagungen / Credits aufgeführt.

Helfen Sie dem Open Source Projekt aktiv durch Ihre Spende!

Der Autor von Tellmatic sagt schon mal Danke. :-)
