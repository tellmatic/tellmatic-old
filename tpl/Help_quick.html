• Einstellungen
    Die Grundeinstellungen wurden bereits bei der Installation vorgenommen und brauchen vorerst nicht geändert zu werden. Stellen Sie jedoch unbedingt sicher das die angegebenen E-Mail-Adressen korrekt sind und prüfen Sie die Logindaten für den SMTP-Server auf Richtigkeit.
  • Adressgruppen anlegen
    Legen Sie nun ein oder mehrere Adressgruppen an. Ein Newsletter richtet sich beim Versand immer an eine oder mehrere Adressgruppen. Eine Adresse wird einer oder mehreren Gruppen zugeordnet. (Beispiel: Gruppe 1: 'Händler' , Gruppe 2: 'Endkunden', Gruppe 3: 'Partner', usw.)
  • Formular erstellen
    Erstellen Sie nun wahlweise ein Formular zur Erfassung der E-Mail-Adressen. Geben Sie hierzu den Namen eines Formulares an und ob die Adressen über Double-Opt-In angemeldet werden sollen (Hier erhält der Benutzer eine E-Mail mit einem Aktivierungslink). Tragen Sie die Feldnamen ein und markieren Sie Feldtyp und Pflichtfelder. Wählen Sie die Gruppe(n) der die angemeldeten Adressen zugeordnet werden sollen.
  • Adressen anlegen - Import, manuelle Eingabe oder per Formular
    Adressen erfassen Sie über die erstellten Anmeldeformulare. Natürlich können Sie Ihren bestehenden Datenbestand auch über die Importfunktion einlesen oder manüll eingeben.
      Import:
      Wählen Sie die Gruppen in welche die importierten Aressen eingetragen werden sollen und die CSV-Datei.
      manuelle Eingabe:
      Geben Sie hier die Adressen manuell ein. Wählen Sie die Gruppe(n) in die die Adresse aufgenommen werden soll.
      über Formulare:
      Rufen Sie hierzu einfach die Onlineansicht des gewünschten Formulars auf.
  • Newslettergruppen anlegen
    Zur besseren übersicht werden Newsletter ebenfalls in Gruppen verwaltet. Diese Gruppen sind jedoch nur intern von Bedeutung und dienen der besseren Aufteilung. Legen Sie beliebige Gruppen für Ihre Newsletter an (Beispiel: Gruppe 1: Preislisten, Gruppe 2: Sonderaktionen, Gruppe 3: allgemeine Rundschreiben, Gruppe 4: Gewinnspiele, usw.)
  • neuen Newsletter erstellen
    Erstellen Sie nun Ihren Newsletter. Geben Sie einen Betreff (Subject) ein - dieser erscheint dann als Betreff in der E-Mail. Fügen Sie ggf. ein Bild ein, hierzu können Sie ein neues Bild hochladen oder per Wysiwyg einfügen. Wählen Sie ggf. eine Html-Datei dessen Inhalt dem Newslettertext angefügt werden soll. Die HTML-Datei kann ebenfalls mit den Variablen versehen werden. Geben Sie ggf. einen Link an der verfolgt (getrackt) wird. Wählen Sie zwischen Massenmailing oder personalisiertem Newsletter.
  • Versand starten
    Wählen Sie einen Newsletter und die Adressgruppen welche den Newsletter erhalten sollen. Wenn Sie das Newsletter zu einem bestimmten Zeitpunkt versenden wollen, geben Sie bitte Datum und Uhrzeit an. Wenn Sie die Versandliste sofort erstellen, ist keine manuelle Aktivierung nötig und der Versand wird sofort aktiviert, andernfalls müssen Sie den Versand für das gewählte Newsletter manuell starten.
  • Statistik
    Über die Listenansichten für Adressen und Newsletter können Sie über das Statistiksymbol die Zusammenfassung für den gewählten Eintrag ansehen. Eine Zusammenfassung finden Sie unter Status
